# 🚀 Orbit of Intelligence

A futuristic **anti-gravity themed e-portfolio** built for **Sinchana M C**, featuring immersive parallax scrolling, floating UI elements, and a cinematic AI-inspired universe.

## 🌌 About the Project

**Orbit of Intelligence** is a modern and interactive e-portfolio designed to create the feeling of floating through a futuristic universe while exploring my academic journey, skills, and projects.

The website combines:

* ✨ Anti-gravity design aesthetics
* 🌠 Cinematic parallax scrolling
* 💎 Glassmorphism UI
* 🚀 Smooth animations
* 📱 Fully responsive design
* ⚡ Optimized performance for GitHub and Vercel deployment

### Run Locally

1. Clone the repository

```bash

```

2. Navigate to the project folder

```bash
cd portfolio
```

3. Open `index.html` in your browser.
